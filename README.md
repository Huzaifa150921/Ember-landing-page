# 🔥 Ember — Premium Restaurant Landing Page

A modern, elegant, and immersive restaurant landing page designed for fine dining restaurants, steakhouses, cafés, and hospitality businesses. Inspired by luxury branding, Ember combines sophisticated typography, smooth animations, and atmospheric visuals to create an unforgettable user experience.

---

## ✨ Features

- 🎨 Premium luxury restaurant design
- 📱 Fully responsive layout
- 🔥 Beautiful hero section with immersive storytelling
- 🌫 Animated ember particle background
- 🍽 Signature menu showcase
- 📸 Interactive gallery section
- ⭐ Customer testimonials carousel
- 📅 Reservation form with validation
- 🕒 Opening hours section
- 📈 Scroll reveal animations
- 🚀 Smooth scrolling navigation
- 📍 Contact & footer section
- ⬆ Back-to-top button
- ⚡ Lightweight and fast loading
- 🎯 Clean and well-structured code
